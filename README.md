## Introduction
SuggestArrHome is a Jellyfin plugin extension for [Home Screen Sections](https://www.github.com/IAmParadox27/jellyfin-plugin-home-sections) which allows users to add the recommended content requested by [SuggestArr](https://github.com/giuseppe99barchetta/SuggestArr) for each user to the web client's home screen.

## Installation

### Prerequisites

- This plugin is based on Jellyfin Version `10.11.2`
- The following plugins are required to also be installed, please following their installation guides:
  - Home Screen Sections (https://github.com/IAmParadox27/jellyfin-plugin-home-sections) at least v2.3.8.0
